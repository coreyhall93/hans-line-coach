How To Use

Scene Run

Walk through scenes like a real rehearsal. Other characters' lines show, then you reveal or type Hans's lines. Perfect for getting the flow down.

Cue & Response

Flashcard-style. See the cue (what's said before your line), then reveal Hans's response. Rate your confidence to track progress.

Type It

The hardest mode. Type Hans's line from memory. Get instant feedback on accuracy.

Quiz

Multiple choice. Great for song lyrics and tricky lines.

Full Script

Read through all Hans scenes with lines highlighted. Tap the amber N dots on Hans lines to see acting notes.

Acting Notes

Character spine, technique, references, and scene-by-scene acting directions. Your rehearsal companion.

Keyboard Shortcuts

  • Space — Reveal line / Next
  • Enter — Submit typed answer
  • 1-3 — Confidence rating (Hard/Good/Easy)

Cloud Sync

Sync your progress across iPhone, iPad, and Mac.

First time? Setup instructions (5 min)
  1. Go to console.firebase.google.com
  2. Click Create a project (name it anything, skip Analytics)
  3. In the left sidebar, click Build → Realtime Database
  4. Click Create Database
  5. Select any location, choose Start in test mode, click Enable
  6. Copy the database URL at the top (looks like https://your-project-default-rtdb.firebaseio.com)
  7. Paste it above and choose any PIN you'll remember
